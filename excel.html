Excel Básico e Avançado

Sobre o curso

Transforme sua relação com os dados através de uma jornada completa pelo Microsoft Excel. Do domínio inicial da interface à automação avançada, você aprenderá a estruturar funções inteligentes e a realizar tratamentos de dados de alto nível. Com foco em resultados, o curso ensina a criar comunicações visuais de impacto com gráficos estratégicos e a dominar o poder das Macros e Tabelas Dinâmicas, transformando processos manuais em sistemas dinâmicos, ágeis e profissionais.

O que você irá aprender

Excel Básico: Apresentar estrutura e conteúdo do curso ● Conhecer o conceito e aplicação do programa Excel Básico ● Identificar os elementos de tela ● Conhecer os operadores mais usados no Excel ● Criar fórmulas básicas ● Criar e formatar planilhas básicas

Excel Dados: Conhecer o conceito de tratamento de dados ● Aplicar recursos para tratamento de dados

Excel Funções: Conhecer a sintaxe das funções ● Aplicar funções para resolução de cálculos ● Identificar códigos de erro das funções ● Criar funções composta

Excel Gráficos: Selecionar dados para representar graficamente ● Escolher modelo de gráfico ● Editar os elementos do gráfico ● Excluir gráfico

Excel Macro: Definir nome para uma macro ● Gravar macro ● Automatizar tarefas mais simples ● Excluir uma macro ● Atribuir macro a botões de formulários e formas

Excel Planilhas Dinâmicas: Inserir tabela dinâmica ● Inserir gráfico dinâmico ● Editar tabela e gráfico dinâmico ● Atualizar tabela dinâmica ● Trabalhar com valores na tabela dinâmica
